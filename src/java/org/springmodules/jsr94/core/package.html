JSR-94 Core beans


springmodules now offers integration with any JSR-94-compliant rule engine.
The Jsr94Template class allows you to easily interact with JSR-94-compliant rule engine in your class; The Jsr94RuleSupport is a convenience superclass that can be used in your business layer implementation. It exposes all methods of Jsr94Template in a similar way to the likes of JdbcDaoSupport.
